Bantuan

Data kehapus ketika install ulang aplikasi?

Pastikan anda sudah membackup/cadangkan datanya bisa pada device storage atau pun google drive. Anda juga bisa mengaktifkan fitur otomatis backup google drive di halaman pengaturan agar data anda aman jadi ketika aplikasi anda terhapus anda bisa memakai fitur restore/pulihkan sehingga data anda bisa kembali.

Mentransfer data dari perangkat lama ke perangkat baru?

Menggunakan fitur cadangkan dan pulihkan data pada halaman pengaturan, pada perangkat lama kita pake fitur cadangkan data terlebih dahulu kemudian di perangkat yang baru kita bisa pake fitur pulihkan.

Data tiba-tiba kehapus?

Kemungkinan pengguna menginstall aplikasi clear cache atau sejenisnya yang secara otomatis menghapus data dari aplikasi catatan keuangan. Pengguna bisa menjaga datanya agar tetap aman dengan menggunakan fitur backup otomatis atau backup secara manual.

Backup terakhir tidak muncul di list google drive?

Kemungkinan file backupnya terlalu banyak solusinya hapus secara manual file backup yang tidak dibutuhkan pada google drive.

Cadangkan dan pulihkan database via google drive tidak bekerja?

Kemungkinan ketika connect ke google drive belum dibuka permission/izin storagenya. Cara membukanya Logout terlebih dahulu kemudian Connect ulang maka akan muncul popup permissionnya dan jangan lupa beri centang izin untuk penyimpanannya.

Iklan masih muncul meskipun sudah mengaktifkan fitur hapus iklan?

Masuk ke halaman hapus iklan/remove ads dan pastikan anda menggunakan akun google yang sama dengan ketika anda membeli fitur hapus iklan.

Dimanakah file hasil export data/laporan berada?

Di internal memory folder /Download atau sesuai info di popup ketika berhasil export.
